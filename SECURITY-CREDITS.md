## Acknowledgement policy

- Reporters are credited with the name or handle they choose. Anonymous
  disclosure is fully supported and respected.
- Crediting happens only after a fix has been merged and an advisory
  (or release-notes entry) is publicly visible. We never publish a
  credit before the fix.
- If a reporter prefers not to be listed publicly, that preference
  overrides everything else in this document.
